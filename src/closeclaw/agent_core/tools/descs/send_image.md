Send an image file to the user. Use this tool when you want to share a generated or existing image with the user.

**Supported formats:** JPEG, PNG, GIF, BMP, TIFF, WebP.

**Limitations:**
- Maximum file size: 10 MB.
- The file must be within the working directory or specified by an absolute path.

**Usage:**
- Provide the `path` to the image file.
- Optionally provide a `caption` (Markdown supported) to describe the image.
